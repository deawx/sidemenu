Basic API documentation

SideMenu Class

Class that represent a list menu. this class is parent for SideMenuMain and SideSubMenu SubClases

new SideMenu([SMItem,..], options)
  • addItem(SideMenu, index)
  • addItems([SideMenu,..], index)

SideMainMenu Class

Extend from SideMenu, represent a master menu dom object.

new SideMainMenu([SMItem,..], options)
  • appendTo(domTarget)

SideSubMenu Class

Extend from SideMenu. SideSubMenu represent a submenu dom object.

new SideMainMenu([SMItem,..], options)

SMSubMenuItem Class

Extend from SMItem, represent a single item with a submenu child.

new SMSubMenuItem("name", [SMItem,..])
  • moveToMenu(SideMenu, index)

SMLinkItem Class

Extend from SMItem, represent a link item.

new SMLinkItem("name", "url", "_blank")

SMButtonItem Class

Extend from SMItem, represent a action button item.

new SMButtonItem("name", onClickHandler)

SMUserAccountItem Class

This is a custom item example. All created custom items have to extend from SMItem.

new SMUserAccountItem("name", "src")
